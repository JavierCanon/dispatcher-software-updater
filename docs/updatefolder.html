Update Data Folder

The Update Data Folder is a local copy of the update data that must be uploaded to all Update Mirrors, so the Updater can update from them.

When you click "Release in Full and Release Update Data" or "Release Update Data Only", this folder is fully erased, and update data is generated into it.

If you checked "Auto-Upload Update Data to Update Mirrors", update data will automatically be uploaded to all Update Mirrors that you configured Auto-Upload URLs for.

Recommended Use: Leave at default location. Allow Dispatcher to Auto-Upload to all your Update Mirrors.

Not Recommended: If you don't Auto-Upload the Update Data found in here after you release updates, you must manually upload ALL changed files from this folder to each of your mirrors. If you forget to upload even a single file, the Updater will be unable to update your software.
